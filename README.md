# 🐦 Flappy Bird Clone

A Flappy Bird clone built using **Python** and **Pygame**.

This project recreates the classic Flappy Bird gameplay where the player controls a bird and tries to fly through randomly generated pipes without colliding.

---

## 🎮 Features

- Smooth bird movement
- Random pipe generation
- Collision detection
- Live score counter
- Welcome screen
- Game Over screen
- Sound effects
- Clean and modular Python code

---

## 🛠️ Built With

- Python 3
- Pygame

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/hardik0069-gif/Flappybird.git
```

Go into the project folder

```bash
cd Flappybird
```

Install dependencies

```bash
pip install pygame
```

Run the game

```bash
python FlappyBird.py
```

---

## 🎮 Controls

| Key | Action |
|------|--------|
| Space | Fly |
| ↑ Arrow | Fly |
| ESC | Quit |
